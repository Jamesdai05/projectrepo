# Introduction of git command

    ## git add
        * The purpose of this command is used for staging the file.It tells Git which file changes you want to include in the next commit.

    ## git rm
        * The purpose of this command is used for remove files from both Git tracking AND your working directory, and stage that removal for the next commit.
    ## git mv
         * The command is used to rename the files and stage the files for the next commit.

    ## git commit
        * The command is used for committing the files changes.

    ## git log
        * The command is used to shows the commit history of your repository — who committed what, when, and with what message.

    ## git restore
        * The command is used to undo changes — either in your working directory or in the staging area.
        * It is a safe undo button in modern Git.
